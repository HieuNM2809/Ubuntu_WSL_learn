Link document: https://learn.microsoft.com/en-us/windows/wsl/install

1. Mở cmd chạy lệnh : wsl --install
2. Khởi động lại máy và Login username, password 
3. Mở WSL : Open cmd or termial -> wsl



Lưu ý : 
 + nếu muốn trở về WSL 1 thì : wsl --set-version Ubuntu 1
 + wsl --status              : check xem đã đặt Window Subsystem for lunix chưa ?
